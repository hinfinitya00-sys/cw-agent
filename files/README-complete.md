# 🤖 CW-Agent 完全自動化システム v2.0

**AI を活用した CrowdWorks 完全自動化システム - 月収100万円達成**

毎日自動で AI 案件をリサーチ・分析・応募まで完全自動化する革新的システムです。

## ✨ システム機能

### 🎯 **完全自動化フロー**
```
📊 案件リサーチ → 🤖 AI評価 → ✍️ 応募文生成 → 📤 自動応募
```

| フェーズ | 機能 | 説明 |
|----------|------|------|
| **🔍 案件検索** | 自動スクレイピング | CrowdWorks から AI 関連案件を自動取得 |
| **🤖 AI評価** | Claude 分析 | 案件の AI 活用可能性・収益性・納品可能性を評価 |
| **✍️ 提案文生成** | 自動ライティング | 案件特化型の魅力的な応募文を AI 生成 |
| **📤 自動応募** | フォーム送信 | 実際の CrowdWorks 応募フォームに自動入力・送信 |
| **📊 結果追跡** | ダッシュボード | リアルタイム進捗・収益・成功率の可視化 |

## 🚀 セットアップ手順

### **前提条件**
- Node.js 16.0+ 
- npm 8.0+
- Chrome/Chromium ブラウザ
- Claude API アカウント

### **1. プロジェクトセットアップ**

```bash
# 1. ファイルを配置
mkdir ~/cw-agent-auto
cd ~/cw-agent-auto

# 2. ダウンロードした3ファイルを配置:
# - server-auto.js
# - cw-agent-complete.html  
# - package.json

# 3. 依存関係インストール
npm install

# 4. データディレクトリ作成
mkdir data
```

### **2. Claude API 設定**

```bash
# Claude API キー取得
# 1. https://console.anthropic.com/ にアクセス
# 2. API Keys → Create Key
# 3. sk-ant-api03-... 形式のキーを取得
```

### **3. システム起動**

```bash
# サーバー起動
npm start

# ✅ 期待される出力:
# 🚀 CW-Agent完全自動化システム起動完了
# 📍 ダッシュボード: http://localhost:3456
# 🤖 自動化機能: 準備完了
```

### **4. ブラウザでアクセス**

```
http://localhost:3456
```

## 🎛️ 使用方法

### **初期設定**

1. **Claude API キー設定**
   - ダッシュボードで「Claude API キー」欄に入力
   - `sk-ant-api03-...` 形式

2. **自動化設定**
   - 実行スケジュール: 毎日 9:00（推奨）
   - 最低 AI スコア: 70点以上（推奨）
   - 1日最大応募数: 5件（推奨）

### **動作テスト**

1. **手動スキャン実行**
   ```
   🔍 手動スキャン ボタン → AI 案件自動検索
   ```

2. **AI評価テスト**
   ```
   リアルタイム案件 タブ → 🤖 AI評価 ボタン
   ```

3. **応募文確認**
   ```
   📝 応募文 ボタン → 生成された提案文をプレビュー
   ```

### **完全自動化開始**

```
▶️ 自動化開始 ボタン → 毎日の完全自動実行開始
```

**自動化により以下が毎日実行されます：**
- 🌅 毎朝9時に AI 案件自動検索
- 🤖 Claude による案件評価・スコアリング
- ✍️ 高評価案件への応募文自動生成
- 📤 応募フォーム自動入力（安全確認あり）
- 📊 結果のダッシュボード更新

## 🆘 トラブルシューティング

### **よくある問題**

**Q: Puppeteer がインストールできない**
```bash
# Chrome を手動でインストール
npm config set puppeteer_skip_chromium_download true
npm install puppeteer
```

**Q: Claude API エラーが発生する**
```bash
# API キー確認
curl -H "x-api-key: YOUR_API_KEY" \
     https://api.anthropic.com/v1/messages
```

**Q: CrowdWorks にアクセスできない**
```bash
# ヘッドレスモードを無効化してブラウザ確認
# server-auto.js 内の headless: false に変更
```

**Q: 案件が取得されない**
```bash
# ネットワーク接続確認
ping crowdworks.jp

# User-Agent 設定確認
# server-auto.js の userAgent 設定を更新
```
